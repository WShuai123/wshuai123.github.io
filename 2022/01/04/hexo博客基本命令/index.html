本地搜索

原理

它用了 Hexo 插件 hexo-generator-searchdb，预先生成了一个文本库 search.xml，然后传到了网站里面。在本地搜索的时候，NexT 直接用 javascript 调用了这个文件，从而实现了静态网站的本地搜索。

安装插件

配置

修改站点配置文件_config.yml，添加以下内容：

  • path：索引文件的路径，相对于站点根目录
  • field：搜索范围，默认是 post，还可以选择 page、all，设置成 all 表示搜索所有页面
  • limit：限制搜索的条目数

然后修改主题配置文件_config.yml：将Local Search部分的enable改为true。

站点地图 sitemap 生成与使用

配置Google Search Console

  • 打开Google Search Console网站
  • 在网址前缀输入你的博客地址。例如，我的就是：https://www.wshuai.xyz/
  • 验证所有权建议使用第二种：HTML标记
  • 复制HTML标记方法里谷歌提供的代码的最后的双引号部分
  • 打开博客的主题配置文件_config.yml
  • 将刚刚复制的代码粘贴到google_site_verification后面（注意冒号之后空一格）
  • 最后保存，再使用hexo命令上传到GitHub
  • 回到Google Search Console，点击验证，即可验证成功
